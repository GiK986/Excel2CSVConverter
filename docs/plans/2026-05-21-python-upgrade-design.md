# Design: Python 3.11 → 3.14.4 Upgrade

**Date:** 2026-05-21
**Scope:** Фаза 1 — само Python version upgrade, без Docker или DB промени

## Цел

Надграждане на Python от 3.11.4 до 3.14.4 и актуализиране на зависимостите в `requirements.txt`. Кодът остава непроменен.

## Промени

### 1. Python среда

Изтриване на старото `venv` и създаване на ново с Python 3.14.4:

```bash
rm -rf venv
python3.14 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. `requirements.txt`

| Пакет | Преди | След | Причина |

| `numpy` | `==1.25.2` | `>=2.0` | numpy 1.x не поддържа Python 3.14 |
| `pandas` | `==2.0.3` | `>=2.2` | pandas 2.0.x не поддържа Python 3.14 |
| `openpyxl` | `==3.1.2` | `>=3.1` | отпушва по-нови patch версии |
| `python-dateutil` | `==2.8.2` | `>=2.9` | актуализация |

### 3. Код

Никакви промени — `converter.py`, `utils.py`, `validators.py`, `logger.py` остават непроменени.
